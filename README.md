# Sistema de E-mail

>Sistema completo de e-mail feito como trabalho para faculdade de Sistemas de Informação

## Funcionalidades

- [ ] F1 - Sistema de cadastro
  - [ ] Nome
  - [ ] Sobrenome
  - [ ] E-mail desejado
  - [ ] Senha
>
- [ ] F2 - Sistema de login
  - [ ] E-mail
  - [ ] Senha
>
- [ ] F3 - Caixa de entrada
  - [ ] Enviar e-mail
    - [ ] Verificar existência destinatário
  - [ ] Listar todos os e-mails
  - [ ] Ler e-mail
  - [ ] Deletar e-mail

## Tecnologias utilizadas

- HTML
- CSS
- JS
- PHP
- MYSQL
